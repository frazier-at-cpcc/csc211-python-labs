CSC-211 · Ethical Hacking with Python I

Core Python Labs

Eight short, interactive labs that build Python from zero — one rung at a time, in the browser. Write real code, run it, and get instant feedback. No install, no account, nothing to submit here.

How it works. Each lab runs a real Python interpreter (Pyodide) entirely in your browser. Type a solution, press Run to see the output, and Check to grade it against the exercise's goals. Work through them in order — each week assumes only what earlier weeks introduced.

The first lab you open downloads the Python runtime (a few seconds); after that it is instant.

Authorized use. These labs teach Python in a security context, but they contain only self-contained language practice — no scanning, no network or system access, and no real targets. The offensive techniques in the course are practiced only inside the college-provided isolated lab environment, against authorized systems. Skill carries the obligation to use it lawfully and with permission.
